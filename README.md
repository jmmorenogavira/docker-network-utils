Network Utilities Container
===========================

Simple container based on Debian with a few network utilities installed:

  - netcat (openbsd version)
  - curl
  - nmap
  - traceroute
  - jq
  - net-tools (arp, ifconfig, netsat, rarp, nameif, route & more)
  - dnsutils
